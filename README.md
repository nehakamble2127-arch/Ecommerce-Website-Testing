# E-Commerce Website Testing Project

## Project Overview
This project is a complete QA testing project for an e-commerce website using SauceDemo.

Website Tested:
https://www.saucedemo.com/

## Testing Types Covered
- Manual Testing
- Automation Testing

## Manual Testing Includes
- Bug Report Documentation
- Test Cases Documentation

## Automation Testing Includes
- Login Test
- Invalid Login Test
- Add to Cart Test
- Remove Product Test
- Checkout Test

## Tools Used
- Python
- Selenium WebDriver
- ChromeDriver
- Microsoft Excel
- VS Code

## How To Run Automation Tests
Open terminal inside Automation_Testing folder:

python test_login.py

Example:
python test_cart.py
